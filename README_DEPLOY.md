# SCHADS GitHub Pages deployment

Upload `index.html` to the root of the `SCHADS` GitHub repository.

Do not upload the zip as the page itself. GitHub Pages serves `index.html`; a zip file will appear as binary/garbled text in a browser.

Recommended steps:
1. Unzip this package.
2. In the GitHub repository, delete or replace any current garbled/binary file being used as the page.
3. Upload the extracted `index.html` to the repository root.
4. Confirm GitHub Pages is set to deploy from the main branch root or the configured Pages folder.
5. Visit https://injecthr.github.io/SCHADS/ after GitHub Pages rebuilds.
